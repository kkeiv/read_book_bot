Пакет, в котором хранятся обработчики апдейтов.

__init__.py	- инициализатора пакета

admin_handlers.py			- Модуль с хэндлерами, срабатывающими на действия пользователя, если он является администратором бота.

subscribers_handlers.py		- Модуль с хэндлерами, срабатывающими на пользователей, купивших подписку

private_user_handlers.py	- Модуль с хэндлерами, срабатывающими на действия пользователей прошедшем регистрацию.

user_handlers.py			- Модуль с хэндлерами для анонимного пользователя, например, для тех, кто первый раз запустил бота.
